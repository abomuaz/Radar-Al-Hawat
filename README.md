# Radar-Al-Hawat

نسخة عرض احترافية من **رادار الحوات** داخل المستودع نفسه، وتشمل الآن:

- توقعات موسعة حتى **16 يومًا**
- خريطة للموقع الساحلي
- تنبيهات لحالة البحر والرياح والموج
- مؤشر صيد ذكي
- إعدادات مخصصة وحدود تنبيه
- دعم PWA للتثبيت على الجوال
- **Offline Mode** لإبقاء آخر البيانات الأساسية محفوظة
- صفحة معلومات/حول التطبيق

## التشغيل محليًا

```bash
cd /workspaces/Radar-Al-Hawat
python3 -m http.server 8000
```

ثم افتح:

```text
http://127.0.0.1:8000/
```

## الملفات الأساسية

- `index.html` — الواجهة الرئيسية
- `styles.css` — التصميم والهوية
- `app.js` — منطق التطبيق والبيانات
- `manifest.webmanifest` — إعدادات الـ PWA
- `service-worker.js` — التخزين المؤقت والعمل كتطبيق مثبت
